# Sales Performance Dashboard (Excel)

This project is an **interactive Excel Sales Dashboard** designed to analyze sales, profit, and customer trends across multiple years, categories, and regions.  
The dashboard provides a clear overview of business performance using KPIs, charts, and slicers.

## Project Objective
- Analyze sales and profit trends over time  
- Compare performance across product categories  
- Track customer growth year-wise  
- Identify top customers and high-performing states  
- Build an interactive and visually appealing Excel dashboard  

## Dashboard Overview
- **Time Period:** 2014 – 2017  
- **Categories:** Furniture, Office Supplies, Technology  

## Key Insights
- **Profit Trend:** Consistent growth observed across all categories, with Technology leading in profitability  
- **Sales by State:** Strong regional performance visible through geographic mapping  
- **Customer Count:** Steady increase in customers year over year  
- **Monthly Sales:** Peak sales observed in the last quarter of the year  
- **Top Customers:** Clear identification of top 5 customers by sales  
- **Category Performance:** Phones and Chairs are top-selling sub-categories  

## Dashboard Features
- Year-wise and Category-wise slicers  
- Interactive charts and visuals  
- Sales and profit trend analysis  
- State-wise sales map  
- Dynamic customer and category insights  

## Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Excel Formulas (SUMIFS, COUNTIFS, IF)  
- Conditional Formatting  
- Data Cleaning and Data Modeling  

## Project Structure
- **Raw Data Sheet** – Original dataset  
- **Processed Data Sheet** – Cleaned and structured data  
- **Analysis Sheet** – Pivot tables and calculations  
- **Dashboard Sheet** – Interactive visualizations  

## Conclusion
This project demonstrates strong **Excel data analysis and dashboarding skills**, showcasing the ability to transform raw data into meaningful business insights.
